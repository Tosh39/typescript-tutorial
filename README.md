## Typescriptのインストール方法
1. 下記コマンドを実行
```
$ sudo npm install -g typescript
```
2. 一度ターミナルのウィンドウを閉じて、再起動する
3. 下記コマンドを実行
```
$ tsc -v
```
バージョンが表示されれば完了です
